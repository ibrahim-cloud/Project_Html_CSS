1/ Qu'est-ce que HTML?

HTML est le langage de balisage standard pour la création de pages Web.

HTML signifie Hyper Text Markup
HTML décrit la structure d'une page Web
HTML est constitué d'une série d'éléments
Eléments HTML indiquent au navigateur comment afficher le contenu
Eléments HTML sont représentés par des balises
Balises HTML étiquette des morceaux de contenu tels que « rubrique », « table » « paragraphe », et ainsi de suite
Les navigateurs ne présentent pas les balises HTML, mais les utiliser pour rendre le contenu de la page

2/ les Balises HTML

Document HTML

Tous les documents HTML doivent commencer par une déclaration de type de document: ''.
Le document HTML lui - même commence par html et se termine par /html.
La partie visible du document HTML est entre body et /body.
